# COILNOVA Website

Website institucional de uma página para a COILNOVA.

## Antes de publicar

1. Abra o ficheiro `index.html`.
2. Procure por `SEU-EMAIL-AQUI`.
3. Substitua pelo email que pretende usar no formulário.
4. Guarde o ficheiro.
5. Publique os ficheiros no GitHub Pages.

## Estrutura

- `index.html` — conteúdo do site
- `styles.css` — design e responsividade
- `script.js` — menu móvel e ano automático
